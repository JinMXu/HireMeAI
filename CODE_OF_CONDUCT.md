# 行为准则

## 我们的承诺

我们希望 HireMe.AI 成为一个友好、尊重、开放的项目。贡献者和维护者都应以耐心、善意和建设性的方式沟通。

## 鼓励的行为

- 尊重不同背景、经验水平和观点。
- 提供具体、建设性的反馈。
- 让讨论聚焦于项目、代码和用户价值。
- 尊重隐私，不在未经允许的情况下分享他人的简历、岗位描述、面试记录、密钥或其他敏感数据。

## 不接受的行为

- 骚扰、威胁、歧视性表达或人身攻击。
- 持续扰乱讨论或恶意消耗维护者精力。
- 未经允许公开他人的私人信息。
- 分享凭证、API Key 或敏感用户数据。

## 执行方式

维护者可以删除不当评论、关闭 Issue、拒绝贡献，或限制违反行为准则的参与者继续参与项目。
