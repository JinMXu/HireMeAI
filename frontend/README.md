# HireMe.AI Frontend

React + TypeScript + Vite frontend for HireMe.AI.

## Development

```bash
npm ci
npm run dev
```

The dev server expects the backend at `http://localhost:8000` and proxies
`/api` through `vite.config.ts`.

## Checks

```bash
npm run lint
npm run build
```

For full setup, privacy notes, and backend instructions, see the repository
root [README.md](../README.md).
